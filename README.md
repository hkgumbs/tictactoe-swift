# How do I ...
?                      |
-----------------------|
run tests              | `make test`
build the executable   | `make build`
run the executable     | `./build/Release/TicTacToe`
play second            | `./build/Release/TicTacToe --second`
clean up               | `make clean`
control stuff          | arrow keys + enter
